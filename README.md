# Smart Management System

A modern, responsive Smart Management System built using React.

## Features
- Dashboard with charts
- Employee & Client Management
- Payment Tracking
- Fully responsive UI
- LocalStorage based data
- Backend-ready architecture

## Tech Stack
- React (Vite)
- React Router DOM
- CSS (No Tailwind)
- Chart.js
